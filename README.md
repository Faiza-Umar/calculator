# calculator 

A responsive and interactive calculator built using **HTML, CSS, and Vanilla JavaScript**.  
This project demonstrates strong fundamentals in DOM manipulation, event handling, and dynamic UI updates without using any external libraries.

---

## Demo

![Calculator Demo](calculator.gif)
---

## Features

- Basic arithmetic operations (+, −, ×, ÷)
- Decimal number support
- Double zero (00) input
- Delete (Backspace) functionality
- All Clear (AC) reset button
- Real-time display updates
- Responsive layout using Flexbox
- Clean and minimal UI

---

## Technologies Used

- HTML5  
- CSS3 (Flexbox)  
- JavaScript (ES6)

## Technical Highlights

- Dynamic button selection using `querySelectorAll()`
- Event-driven programming with `addEventListener()`
- Conditional logic for delete, clear, and evaluation
- Expression evaluation using JavaScript

## Purpose

This project was built to strengthen core front-end development skills and demonstrate the ability to build interactive web applications using pure JavaScript.
